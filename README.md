### 运行指导
1. 直接运行 com.hualuo.Main#main 方法
2. 输入k值[1,2,3]
3. 点击“选择图像”
4. 输入待隐藏信息
5. 点击"隐藏"，得到两张图片
6、点击提取，从两张图片中获取隐藏的信息
